How to deploy each side
GitHub side:

Edit the file on github.com (or push from your local copy)
GitHub Pages auto-publishes within ~1 minute
Users may need to refresh their browser to see changes

Laptop side:

Edit the file in C:\Users\mingy\cleanhome-backend\functions\
Run firebase deploy --only functions in your terminal
Takes 2–3 minutes, then it's live on Google
